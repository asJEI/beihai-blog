# Beihai Blog - WordPress主题

一个简约现代的浅色WordPress主题
效果预览：http://www.hokkai2005.online

## 特性

- **简约现代设计**：浅色背景，清晰的排版，舒适的阅读体验
- **固定导航栏**：始终显示在页面顶端，带有毛玻璃效果
  - 左侧：站点Logo
  - 中间：主导航菜单
  - 右侧：站内搜索栏
- **智能问候语横幅**：根据东八时区（北京时间）显示"上午/下午/晚上好"问候语
- **作者信息悬浮框**：左侧圆角设计，可点击展开/收起，展示：
  - 作者头像、名称、简介
  - 随笔栏（个人签名）
  - **日间/夜间模式切换开关**
- **完全响应式**：完美适配桌面端、平板和手机
- **可自定义背景**：横幅背景图片可在主题自定义器中设置
- **现代评论区**：
  - 磨砂玻璃效果的圆角评论卡片
  - 日期时间以小字显示在评论右下角
  - 嵌于文本框内的圆角发送按钮
  - 作者徽章标识
  - **全端自适应**：完美适配桌面、平板、手机及触控设备
- **性能优化**：轻量级代码，快速加载

## 日间/夜间模式切换开关设计

主题包含一个精美的胶囊轨道式主题切换开关：

- **胶囊轨道设计**：现代化圆角外观
- **日间模式**：
  - 轨道从右至左暖光渐变（金色/琥珀色）
  - 滑块位于左侧，显示太阳图标 ☀️
  - 滑块采用立体设计，高光质感
- **夜间模式**：
  - 轨道从左至右暗色渐变（深蓝/灰色）
  - 滑块位于右侧，显示月亮图标 🌙
  - 滑块切换为深色风格
- **动画效果**：
  - 弹性动画过渡（cubic-bezier曲线）
  - 700ms平滑过渡
  - 滑块立体光影变化
  - 图标淡入淡出切换

## 安装

1. 将整个 `beihai-theme` 文件夹复制到 WordPress 的 `wp-content/themes/` 目录
2. 登录 WordPress 后台
3. 进入 **外观 → 主题**
4. 找到 "Beihai Blog" 主题并点击 **启用**

## 设置

### 横幅区域设置

1. 进入 **外观 → 自定义**
2. 找到 **横幅区域设置** 部分
3. 可以设置：
   - **横幅背景图片**：建议尺寸 1920x600 像素
   - **横幅标题**：自定义欢迎标题
   - **横幅描述**：自定义副标题描述
   - **显示时间问候语**：开关问候语显示

### 作者信息悬浮框设置

1. 进入 **外观 → 自定义**
2. 找到 **作者信息悬浮框** 部分
3. 可以设置：
   - **显示作者信息悬浮框**：开关悬浮框显示
   - **作者名称**：显示的名称
   - **作者头像**：上传头像图片（建议 200x200 像素）
   - **作者简介**：简短的介绍文字
   - **随笔栏（个人签名）**：座右铭或个性签名

### 菜单设置

1. 进入 **外观 → 菜单**
2. 创建菜单并分配到以下位置：
   - **主导航菜单**：页面顶部的导航栏（中间位置）
   - **页脚归档菜单**：页脚左侧区域（显示文章归档链接）
   - **页脚站外链接菜单**：页脚右侧区域（显示站外链接）

### 页脚设置

1. 进入 **外观 → 自定义 → 页脚设置**
2. 可以设置：
   - **备案信息HTML**：输入ICP备案号、公安备案号等HTML代码
   - **GitHub链接**：GitHub个人主页链接
   - **Twitter链接**：Twitter/X个人主页链接
   - **微博链接**：微博个人主页链接
   - **自定义RSS链接**：如不填写则使用默认RSS

### Logo设置

1. 进入 **外观 → 自定义 → 站点身份标识**
2. 上传您的Logo图片

## 响应式适配

主题完全响应式设计，适配各种屏幕尺寸：

- **大屏桌面**（1200px+）：完整三栏布局，左侧悬浮作者信息框
- **桌面**（1024px-1200px）：优化间距和字体大小
- **平板**（768px-1024px）：桌面端布局
- **手机**（480px-768px）：
  - 搜索栏变为搜索按钮，点击展开搜索面板
  - 导航菜单折叠为汉堡菜单
  - 作者信息悬浮框移至左下角
- **小屏手机**（<480px）：进一步优化间距和字体

## 问候语时间规则

主题根据东八时区（北京时间）自动判断并显示：

- **上午好** ☀️（5:00 - 11:59）
- **下午好** 🌤️（12:00 - 17:59）
- **晚上好** 🌙（18:00 - 4:59）

## 主题模式

- **日间模式**：浅色背景，深色文字，暖色调强调色
- **夜间模式**：深色背景，浅色文字，蓝色调强调色
- **自动切换**：支持跟随系统偏好自动切换
- **记忆功能**：记住用户的选择，下次访问时自动应用

## 文件结构

```
beihai-theme/
├── style.css              # 主题样式（响应式 + 夜间模式 + 评论区）
├── functions.php          # 主题功能（含自定义器设置）
├── index.php              # 首页模板
├── header.php             # 头部模板（导航栏、搜索栏、作者悬浮框）
├── footer.php             # 页脚模板（三栏布局：归档、备案、站外链接）
├── single.php             # 单篇文章模板
├── page.php               # 页面模板
├── archive.php            # 归档/分类模板
├── search.php             # 搜索结果模板
├── searchform.php         # 搜索表单
├── comments.php           # 评论模板（现代圆角风格）
├── 404.php                # 404页面
├── assets/
│   └── js/
│       └── main.js        # JavaScript交互（含搜索、作者悬浮框、主题切换）
├── inc/
│   └── template-tags.php  # 模板标签函数
└── README.md              # 说明文档
```

## 页脚布局

页脚采用三栏布局设计：

- **左侧（归档菜单）**：显示文章归档链接，自动显示最近6个月的归档
- **中间（备案信息）**：空置HTML代码区，可在自定义器中填入备案信息HTML
- **右侧（站外链接）**：显示站外链接菜单 + GitHub/Twitter/微博/RSS社交图标链接

## 兼容性

- WordPress 5.8+
- PHP 7.4+
- 现代浏览器（Chrome, Firefox, Safari, Edge）
- 支持深色模式（跟随系统设置或手动切换）
- 支持高DPI/Retina显示屏
